# Commands

## Basic Commands

Check opshit
```bash
node opshit
```

Check opshit version
```bash
node opshit -v
```

Simply run once
```bash
node opshit serve
```

Run continuously (Required nodemon globally)
```bash
nodemon opshit serve
```

## File Creation Commands

Create a new controller
```bash
node opshit make:ctrl <ControllerName>
```
